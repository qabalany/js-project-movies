# Movies App

A React movie app using TMDB API with React Router for navigation.

## Live Demo
https://movies.fartist.live
https://moviesass.netlify.app/
## Features

### Requirements (G)
- [x] Multi-page app with React Router
- [x] Home page with popular movies list
- [x] Movie detail page with full info
- [x] Responsive design (320px - 1600px)
- [x] Accessible (alt text, aria-labels, keyboard nav)

### Stretch Goals (VG)
- [x] 404 Not Found page
- [x] Loading spinner component
- [x] Category dropdown (popular/upcoming/top rated/now playing)
- [x] Genre filter links

## Tech Stack
- React 19
- React Router
- Vite
- TMDB API

## Run Locally
```bash
npm install
npm run dev
```
